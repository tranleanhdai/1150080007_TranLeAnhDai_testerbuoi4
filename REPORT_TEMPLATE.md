# Báo cáo kiểm thử – Organization Management

## 1. Giới thiệu
- Mục tiêu: xây dựng Form quản lý Organization và kiểm thử theo yêu cầu đề.
- Chức năng chính: nhập thông tin Organization, kiểm tra hợp lệ, lưu DB, mở Director Management.

## 2. Môi trường & công cụ
- Ngôn ngữ: C# (.NET Framework 4.7.2)
- UI: WinForms
- Database: SQLite (System.Data.SQLite)
- Unit test: MSTest
- IDE: Visual Studio (khuyến nghị 2019+)

- Xem file `TESTCASES.md` (>= 15 test case).
